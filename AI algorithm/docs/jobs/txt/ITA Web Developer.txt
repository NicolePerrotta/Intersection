Web Developer
Siamo alla ricerca di un eccellente sviluppatore web che sia responsabile della codifica, del
design innovativo e del layout dei siti web dei clienti. Le responsabilità dello sviluppatore web
includono la costruzione del nostro sito web dall'idea fino al completamento, con un approccio
bottom-up, modellando tutto, dalla home page al layout e alle funzioni del sito.
La maggior parte del lavoro di sviluppo web avverrà tramite CMS utilizzando Drupal e
Wordpress.
